# L00 任务说明

请完成以下任务。

## 任务 1：检查基础工具

运行：

```bash
python3 --version
git --version
```

如果失败，请记录原因。

## 任务 2：启动 Coding Agent

至少启动一种工具：

- opencode
- Cursor
- Codex

在报告中记录：

- 工具名称
- 版本号或安装方式
- 模型配置是否完成
- 是否能正常发起一次请求

不要记录 API key。

## 任务 3：完成一次基础体验

任选一种：

1. 让 Coding Agent 解释本文件。
2. 让 Coding Agent 阅读 `README.md` 并总结任务。
3. 让 Coding Agent 生成一个后续练习计划。

记录你的 prompt 和输出摘要。

## 任务 4：下载或安装一个示例 skill

如果当前工具支持 skill，请下载或安装一个示例 skill。

如果无法安装，请记录：

- 失败原因
- 报错摘要
- 你尝试过的解决方式

## 任务 5：填写报告

在 `agent_setup_report.md` 中填写以下章节：

```markdown
# Agent Setup Report

## Tool

## Model Configuration

## Basic Prompt Trial

## Skill Installation

## Problems

## Next Actions
```
