# L00 讲师备注

## 讲师目标

本课不要讲成工具宣传或 prompt 技巧大全。重点是让每位学员具备后续实战课的最低环境条件。

## 建议演示

1. 从空终端启动一个 Coding Agent。
2. 配置模型服务。
3. 让 agent 阅读一个小文件。
4. 要求 agent 先计划再修改。
5. 运行一次测试或简单命令。
6. 展示一个 skill 下载或安装过程。

## 重点提醒

- API key 和内部地址不能写进仓库。
- prompt 技巧不需要堆很多，后续课程会在真实任务中反复训练。
- `context.md` 是后续所有课程的共同要求。

## 课堂收集

建议收集：

- 哪些工具安装失败。
- 哪些模型服务配置最容易出错。
- 哪些代理问题最常见。
- 哪些 skill 下载方式不可用。
