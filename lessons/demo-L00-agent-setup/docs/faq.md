# L00 FAQ

## Q1: 必须同时安装 opencode、Cursor 和 Codex 吗？

不必须。第一期建议至少安装一种团队推荐工具。安装多种工具有助于比较差异，但不是通过条件。

## Q2: API Key 应该写在哪里？

按工具要求写到本地配置或环境变量中。不要写入本仓库，不要提交到 `context.md` 或 `agent_setup_report.md`。

## Q3: 如果模型服务连不上怎么办？

请记录：

- 使用的工具
- 配置项名称，不包含密钥
- 报错摘要
- 是否使用代理
- 你尝试过的排查步骤

课堂会集中答疑。

## Q4: L00 的验证脚本会检查真实模型调用吗？

默认不检查。模型服务和权限环境差异较大，验证脚本只检查基础命令和必要报告文件。

## Q5: 下载 skill 失败算不算不通过？

如果是网络、权限或工具版本问题，不直接算不通过。但需要在报告中记录原因、尝试过程和后续解决计划。
